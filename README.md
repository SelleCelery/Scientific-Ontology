# Scientific Ontology / 存在論科学

Scientific Ontology is a conceptual framework for describing existence, history, boundary, communication, and ethics within a shared structural vocabulary.

存在論科学は、存在・履歴・境界・通信・倫理を、共通の構造語彙で記述するための概念フレームワークです。

This repository is the public interface of the project.  
It provides maps, introductions, glossary entries, and application notes.  
Core files and implementation-adjacent materials are maintained separately.

このリポジトリは、プロジェクトの公開用インターフェースです。  
体系マップ、入門、用語集、応用ノートを置きます。  
中核ファイルおよび実装近接資料は別管理とします。
