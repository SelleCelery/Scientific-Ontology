# Private Core Not Included

This public repository contains the public interface and introductory materials of the Scientific Ontology project.

Core axioms, full implementation schemas, private drafts, personality runtime files, and detailed Role-AP / Boundary Heat Log specifications are maintained separately and are not included in this repository.

この公開リポジトリには、存在論科学プロジェクトの公開用インターフェースと入門資料のみを置きます。

中核公理、完全な実装スキーマ、非公開草案、人格ランタイムファイル、Role-AP / Boundary Heat Log の詳細仕様は別管理とし、このリポジトリには含めません。
